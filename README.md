# Movies Web Application  

## 📌 Project Overview

The Movies Web Application is a responsive front-end project developed as part of my graduation requirements.  
The application enables users to explore movies, view detailed information, register, log in, and manage a personalized watchlist.

This project was built using pure front-end technologies without relying on modern JavaScript frameworks.

---

## 🎯 Project Objectives

- Apply front-end development fundamentals in a practical project  
- Practice DOM manipulation and dynamic rendering  
- Implement authentication logic using LocalStorage  
- Integrate external APIs using Fetch API  
- Build a fully responsive user interface  

---

## ✨ Key Features

- Movie search functionality  
- Dynamic movie details page  
- User registration and login system  
- Client-side form validation  
- Add / Remove movies from Watchlist  
- Data persistence using LocalStorage  
- Responsive layout using Bootstrap  

---

## 🛠️ Technologies Used

- **HTML5** – Semantic page structure  
- **CSS3** – Styling and layout design  
- **Bootstrap 5** – Responsive grid system  
- **JavaScript (ES6)**  
  - DOM Manipulation  
  - Event Handling  
  - Form Validation  
  - LocalStorage Management  
  - Fetch API Integration  

---

## 🔐 Authentication System

- Users can create an account through the Sign Up page  
- User data is stored in LocalStorage  
- Login system validates stored credentials  
- Navigation is controlled based on login state  

---

## ❤️ Watchlist System

- Users can add movies to their personal watchlist  
- Movie IDs are stored in LocalStorage  
- Watchlist page dynamically renders saved movies  
- Users can remove movies at any time
